Caríssima Comunidade Java Cearense,

O CEJUG inicia a década com um novo sistema que promete coletar, organizar e transformar em conhecimento as informações geradas e compartilhadas pela comunidade.

O primeiro objetivo que queremos alcançar é a definição do que é ser realmente um membro do CEJUG. Atualmente adotamos o conceito de que para se tornar membro do CEJUG é preciso simplesmente se registrar na lista de discussão principal. No entanto, temos percebido que mesmo pessoas não-técnicas, como empresários, e aqueles que decidiram cancelar a inscrição por causa do excesso de emails, gostariam de manter contato com o grupo, não necessariamente entrando em discussões técnicas, mas ajudando de outras formas. Com a adoção de uma aplicação separada da lista de discussão, podemos manter o status de membro dessas pessoas.

Outro objetivo relevante é se tornar independente de aplicações de terceiros. O processo de migração do java.net para a infra-estrutura Kenai nos custou vários dias sem nenhum controle sobre as listas de discussão, a impossibilidade de adicionar novos membros e a indisponibilidade por tempo indeterminado do histórico da lista. Além de tudo isso, continuamos sem acesso aos dados gerenciados pelo projeto Kenai, e por conseguinte, sem acesso aos dados relevantes produzidos pela comunidade. Desenvolvendo nossa própria solução podemos fazer com que os dados trabalhem para nós e o grupo consiga se organizar para crescer ordenadamente.

Finalmente, gostariamos de gerar mais uma fonte de conhecimento para o grupo. A aplicação é totalmente de código aberto e integralmente disponibilizada no SVN do Kenai. Todos podem executar e ver como o CEJUG funciona. Os iniciantes terão uma fonte sólida para iniciar seus estudos no desenvolvimento de aplicações web com Java. Os mais experientes poderão contribuir com correção de bugs, refactoring, e desenvolvimento de novas funcionalidades de acordo com o planejamento no issue tracking do projeto. Os design patterns adotados no projeto podem ser objeto de valiosas discussões na comunidade. Enfim, trata-se de um grande diferencial para o grupo.
